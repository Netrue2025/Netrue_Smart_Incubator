# Smart AI Incubator V2 Frontend

React, Vite, TypeScript, TailwindCSS, shadcn-style local UI primitives, Zustand, React Router, Axios, and Recharts.

## Run

```powershell
npm install
npm run dev
```

Open `http://localhost:5173`. The Vite dev server proxies `/api` and `/ws` to the backend on port `8000`.
